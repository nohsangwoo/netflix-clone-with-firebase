# Netflix clone

> TMDB API를 사용하여 영화의 data를 받아옴(title, poster, date, etc....)
> data를 받아올때 axios를 사용하며 방법을 학습
> DB사용 안함
> movie-trailer 라이브러리를 사용하여 영화 poster 클릭시 해당 영화의 트레일러가 재생되게 구현
> firebase에 호스팅
